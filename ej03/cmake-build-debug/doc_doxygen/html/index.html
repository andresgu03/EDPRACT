Introducción

El juego de las letras
En esta práctica querremos representar el famoso juego del Scrabble. Este juego consiste en formar palabras de un idioma dadas ciertas letras. Cada letra lleva asociada una puntuación y el objetivo es , o bien conseguir el mayor número de puntos posibles, o bien conseguir la letra más larga posible.
Para ello necesitaremos tener una bolsa con letras, donde cada letra irá acompañada de su puntuación asociada. También deberemos tener un Diccionario, para saber si las palabras que están siendo formadas son o no válidas.

Ejecutables

  1. conjunto_letras.cpp
    • Recibe dos argumentos, la ruta al archivo de entrada con información sobre el conjunto de letras y la palabra a la que se le ha de calcular la puntuación.
      • El programa imprimirá por pantalla la puntuación de la palabra( siendo esta la suma de todas las puntuaciones de las letras de la palabra).
      • Se mostrarán, si es necesario, los correspondientes errores de lectura del fichero de entrada.
  2. palabras_longitud.cpp
    • Recibe dos argumentos, un archivo con las palabras de un diccionario y un entero con la longitud de las palabras que buscamos.
      • El programa construirá un objeto de la clase Dictionary con las palabras del fichero de entrada.
      • Extraerá del diccionario las palabras con la longitud deseada y las imprimirá por pantalla.
      • Se mostrarán, si es necesario, los correspondientes errores de lectura del fichero de entrada.
  3. cantidad_letras.cpp
    • Recibe dos argumentos, un archivo con las palabras de un diccionario y un fichero de letras.
      • El programa construirá un objeto de la clase Dictionary con las palabras del fichero de entrada y un LettersSet con el fichero de letras.
      • Imprime por pantalla cada letra del LettersSet, el número de ocurrencias de cada una en el diccionario y su frecuencia relativa.
      • Se mostrarán, si es necesario, los correspondientes errores de lectura del fichero de entrada.
